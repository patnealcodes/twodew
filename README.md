# bootstrap-angular
A base bootstrap project set up with Angular.js. Features karma &amp; jasmine unit testing.

## includes:

### node modules
* bower
* grunt
* grunt-contrib-connect
* grunt-contrib-jshint
* grunt-contrib-sass
* grunt-contrib-watch
* jasmine
* karma
* karma-chrome-launcher
* karma-jasmine
* karma-phantomjs-launcher
* phantomjs

## bower components
* angular
* angular-mocks
* angular-route
* angular-ui-router
